**属性（Property，方括号包裹着的）** 和  **元素属性（Attribute）** 的区别在于，Attribute 只是用于初始化，而 Property 是绑定值，会随着数据源的变化而变化，也就是实时更新
>这句话值得再强调一次： 模板绑定是通过 property 和事件来工作的，而不是 attribute。
